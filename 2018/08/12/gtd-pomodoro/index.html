A great part of most organizational systems like GTD is focused on destilling the next actions based on all input one gets. Completing those actions should realize the most value possible at this very point in time for each unit of time invested. You start capturing, processing processing, organizing and prioritizing all tasks into one big beautiful next action list.

The caveat of those organizational systems is this: the only moment when value is realized is when tasks get done. All other activities managing your organizational system have negative value. It costs time, and inherently has no value.

The question is, now what? If you are like me, actually putting in the work is not something that comes in naturally. To really start tackling the next actions, I devised a process which removes any ambiguity and uncertainness from this execution phase; freeing up mental resources for actually executing the tasks that brings in the most value.
